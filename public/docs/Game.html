players array An array of players, see Player class.
hands Array.<Object> The current hand since the game was started.
round number The current round in this hand. A round is counted after every player has made a move. Rounds start counting from 1, a round of 0 indicates that the hand has not started and new players can potentially enter the game. The round will be used to select the 'dealer' for the next hand.
