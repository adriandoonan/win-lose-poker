WIN LOSE POKER

A game that simulates poker with a dealer, a user player and a couple of computer players.

Big Dreams

  • Intelligent opponents
  • Learning mode
  • Minigames to win back your money

MVP

  • Bots will always match current bet
  • It's limit poker and no reraise
  • Nobody gives up
  • Hand data is in the console
    • We're web developers now, get used to having it open 😜

Custom HTML Elements

						
							export class SpinningCard extends HTMLElement {
								static observedAttributes = ["spin"];
							  
								constructor() {
								  super();
								  this.spin = 0;
								}
							  
								connectedCallback() {
									selfDestruct()
								}
								attributeChangedCallback(name, oldValue, newValue) {
									doSomething(newValue)
								}
							}
							
							customElements.define("spinning-card", SpinningCard);
